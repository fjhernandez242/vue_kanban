# Panel de administración de tareas tipo Kanvan

Una aplicación para la administración de tareas, conteniendo además una sección
para la administración de empleados de los cuales se podrá agregar información
como puesto y correo electrónico, ayudando a delegar las tareas enviando las
actividades vía correo electrónico una vez generadas en el panel.

![Panel](https://github.com/fjhernandez242/vue_kanban/blob/main/vue_kanban.jpeg)

## Mejoras

* Completar la actualización de tareas.
* Integrar sección CRUD par empleados.
* Envio de tareas por correo.
